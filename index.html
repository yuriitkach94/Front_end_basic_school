Список команд при роботі з терміналом

До галереї
  • git commit - зареєструвати зміни у локальний репозиторій
  • git branch - показати всі локальні та віддалені (remote) гілки
  • git checkout -b branch name - створити нову гілку в локальному репозиторію та перейти на цю гілку
  • cd - змінити директорію
  • git add - додати файли у локальний репозиторій
  • git status - переглянути наявність усіх файлів у локальному репозиторію та
  • git push - покласти зміни з локального репозиторію у віддалений
  • git pull - оновити дані з віддаленого репозиторію по поточній гілці
  • git checkout branch name - змінити гілку
  • git remote - показує віддалено налаштовані сервери
  • git clone - клонування репозиторію до комп'ютера
